Class FabrikBone2D

Кость.

FabrikBone2D состоит из начальной и конечной позиции и FabrikJoint2D, который ограничивает возможность поворота относительно предыдущей кости в цепи или мировой системы координат.

Private _joint

_joint: FabrikJoint2D = new FabrikJoint2D()
  • Defined in main/FabrikBone2D.ts:23

_joint Сустав прикреплённый к FabrikBone2D.

Каждая кость содержит один сустав, который крепит данную кость к предыдущей.

По умолчанию сустав ничем не ограничен. Он может вращаться во всех возможных направлениях. Изменить параметры сустава можно через конструктор кости или с помощью методов {@link #setClockwiseConstraintDegs(float)} и {@link #setAnticlockwiseConstraintDegs(float)}.
