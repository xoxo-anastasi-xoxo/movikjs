Static getConstrainedUV

  • getConstrainedUV(directionUV: Vec2f, baselineUV: Vec2f, clockwiseConstraintDegs: number, antiClockwiseConstraintDegs: number): Vec2f
    • Defined in utils/Vec2f.ts:281

    Возвращает единичный вектор с примененными к нему ограничениями. Если угол между идеальным положением вектора и бейзлайном привышает ограничения, то возвращаем вектор, который повернут на граничный угол относительно бейзлайна. В противном случае - возвращаем идеальный вектор.

    Parameters

    • directionUV: Vec2f

      Идеальное положение вектора, которое возможно недостижимо.

    • baselineUV: Vec2f

      Вектор относительно которого считаются углы.

    • clockwiseConstraintDegs: number

      Угол-ограничение по часовой стрелке.

    • antiClockwiseConstraintDegs: number

      Угол-ограничение против часовой стрелки.

    Returns Vec2f
