Class FabrikBone3D

Кость.

FabrikBone3D состоит из начальной и конечной позиции и FabrikJoint3D, который ограничивает возможность поворота.

Private _boneConnectionPoint

_boneConnectionPoint: BoneConnectionPoint = BoneConnectionPoint.END
  • Defined in main/FabrikBone3D.ts:22

Если данная кость присоединена к кости из другой цепи, к какому концу другой цепи она должна присоединяться?

По умолчанию присоединяется к концу данной кости.

Это поле может быть изменено с помощью метода {#link #setBoneConnectionPoint(BoneConnectionPoint)}, или при соединении кости с другой цепью с помощью метода {@link FabrikStructure3D#connectChain(FabrikChain3D, number, number, BoneConnectionPoint)}.

Private _joint

_joint: FabrikJoint3D = new FabrikJoint3D()
  • Defined in main/FabrikBone3D.ts:33

_joint Сустав для данной кости.

У каждой кости есть один сустав, располагающийся в начальной позиции кости. Он определяет как кость может быть повернута относительно предыдущей кости в цепи.

По умолчанию сустав не имеет ограничений, но они могут быть заданы с помощью методов {@link #setClockwiseConstraintDegs(number)} и {@link #setAnticlockwiseConstraintDegs(number)}.
