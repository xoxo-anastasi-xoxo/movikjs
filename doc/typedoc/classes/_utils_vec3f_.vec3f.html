constructor

  • new Vec3f(value?: number | Vec3f, y?: number, z?: number): Vec3f
    • Defined in utils/Vec3f.ts:18

    Конструктор, который принимает четыре вариации параметров:

    Без параметров - все поля принимают значения по умолчанию.

    Конструктор копирования. Создается глубокая копия объекта.

    Parameters

    • Optional value: number | Vec3f

      Координата вектора по каждой из осей.

      Конструктор с тремя параметрами - значения координат по X, Y, Z осям соответственно.

    • Optional y: number

      Значении координаты по оси Y.

    • Optional z: number

      Значении координаты по оси Z.

    Returns Vec3f

approximatelyEquals

  • approximatelyEquals(v: any, tolerance: any): boolean
    • Defined in utils/Vec3f.ts:123

    Проверяет вектора на примерное равенство. Векторы примерно равны если модуль их координат по каждой из осей меньше tolerance.

    Если значение tolerance меньше 0 - бросаем ошибку.

    Parameters

    • v: any

      Вектор для сравнения.

    • tolerance: any

      Значение "терпимости" к разнице.

    Returns boolean

    Результат сравнения.
